#### First project in data analysis club
- This project was maintained by 오서영, [허지혜](https://github.com/jihyeheo), [이수빈](https://github.com/I-SUBIN), [강수연](https://github.com/Kangsooyeon)

## Dataset
- BUSAN_gas_consumption dataset
- BUSAN_average_tempurature : ta_20200329144537.csv

## Web page

1. Linear Regression
2. Linear Regression with Gradient Descent
